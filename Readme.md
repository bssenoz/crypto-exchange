# Kripto Borsam

## Clone project
```
git clone https://github.com/bssenoz/crypto-exchange
cd crypto-exchange
```

## Project setup
```
yarn install
```

## Project start
```
npx expo start
```

## Set environment variables
Copy [sample.env](./sample.env) file then changes values.
```
cp sample.env .env
```

## Admin Information
```
admin@gmail.com || admin123
```
